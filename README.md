# 🚀 Smart Log Analyzer (Spring Boot + Kafka + AI)

## 📌 Overview
Smart Log Analyzer is a scalable backend system that processes application logs using Kafka and provides AI-based error explanations to reduce debugging time.

---

## 🔥 Features
- 📂 Upload log files for processing
- ⚡ Kafka-based asynchronous log ingestion
- 🧵 Async processing for handling large log volumes
- 💾 Store logs in database using Hibernate (JPA)
- 🤖 AI-powered error explanation using OpenAI API
- 🐳 Dockerized application for easy deployment
- ☸️ Kubernetes-ready configuration

---

## 🧠 Tech Stack
- Java
- Spring Boot
- Apache Kafka
- Hibernate (JPA)
- MySQL
- OpenAI API
- Docker
- Kubernetes
